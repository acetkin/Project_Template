# Task Board
This document tracks the status of active tasks.
It represents the current operational state of the project.

## Board Rules
- Only tasks already listed in Task_Backlog.md may appear here.
- A task may exist in only one column at a time.
- Tasks must not be added directly without approval.
- Moving a task does not imply a decision.

## Board Columns

### To Do
Tasks approved to be worked on but not yet started.

### In Progress
Tasks currently being worked on.

### Blocked
Tasks that cannot proceed due to unresolved dependencies or questions.

### Done
Tasks completed according to their definition of done.
